17. Массив квадратов

Сравните два массива. Элементы массива b должны быть квадратами элементов массива a. Элементы в массивах могут быть расположены в любом порядке.

Пример массивов:

a = [121, 144, 19, 161, 19, 144, 19, 11]
b = [121, 14641, 20736, 361, 25921, 361, 20736, 361]

Напишите функцию comp(a, b), которая вернёт true, если:

a = [121, 144, 19, 161, 19, 144, 19, 11]
b = [11*11, 121*121, 144*144, 19*19, 161*161, 19*19, 144*144, 19*19]

или вернётся false, если элементы одного массива не являются квадратами другого:

a = [121, 144, 19, 161, 19, 144, 19, 11]
b = [132, 14641, 20736, 361, 25921, 361, 20736, 361]

(так как 132 не является квадратом ни одного элемента из массива a)
Решение на JavaScript 
a = [121, 144, 19, 161, 19, 144, 19, 11];
b = [121, 14641, 20736, 361, 25921, 361, 20736, 361];
comp = (a, b) => {
    a = a.sort((x, y) => (x - y));
    b = b.sort((x, y) => (x - y));
    for (i = 0; i < a.length; i++) {
        if (Math.pow(a[i], 2) != b[i]) {
            return false
        }
    }
    return true;
}
console.log(comp(a,b));

23. Очередь в супермаркете

В супермаркете большая очередь на кассах. Вам нужно написать функцию, которая рассчитает время, необходимое покупателями для оплаты покупок.

В функцию поступает две переменные:
  • customers — массив положительных целых чисел, представляющий очередь. Каждый элемент массива означает сколько времени необходимо покупателю для оплаты товара;
  • N — число касс
Функция должна вернуть, сколько времени потребуется всем покупателям для оплаты товара.

Существует всего одна очередь и несколько касс. Покупатели идут в порядке очереди. Как только одна из касс освобождается, туда идёт следующий покупатель из очереди.

Например:

queue_time([5,3,4], 1) == 12
// так как у нас одна касса, 5 + 4 + 3 = 12

queue_time([10,2,3,3], 2) == 10
// так как у нас две кассы и пока первый покупатель будет стоять в первой кассе, второй, третий и четвёртый уже оплатят товар (2 + 3 + 3)

queue_time([2,3,10], 2) == 12
// первый покупатель идёт в первую кассу, второй – во вторую. Первая касса освобождается быстрее второй и туда идёт третий покупатель. Поэтому на первой кассе время обслуживания = 2 + 10 = 12, а на второй = 3
Решение на JavaScript 
queue_time = (a, n) => {
    cashboxes = new Array(n).fill(0);
    for (elem in a) {cashboxes[cashboxes.indexOf(Math.min(...cashboxes))] += a[elem]};
    return (Math.max(...cashboxes));
}
console.log(queue_time([10,2,3,3], 2));

Например, корректная контрольная строка будет выглядеть как aaabbbbhaijjjm, что значит, что принтер три раза использовал букву a, четыре раза букву b и так далее. Вам нужно написать функцию printer_error, которая по входной строке выдаст частоту ошибок принтера, которая вычисляется как отношение количества ошибок к длине контрольной строки. Не сокращайте данную строку!

Контрольная строка может состоять только из букв от a до z. Например:

error_printer("aaabbbbhaijjjm") == 0/14
error_printer("aaaxbbbbyyhwawiwjjjwwm") == 8/22
Решение на JavaScript 
error_printer = a => {
    errors = 0;
    for (elem in a) {a[elem] > 'm' ? errors += 1 : 0};
    console.log(errors);
}
error_printer('aaaxbbbbyyhwawiwjjjwwm');

27. Подпрыгивающие мячи

Ребёнок играет с мячиком на n-ом этаже высотного здания. Высота этого этажа h известна. Он бросает мячик из окна. Мяч подпрыгивает, например, на две-трети высоты h (bounce = 0.66). Его мама смотрит в окно на высоте 1.5 метра от земли. Как много раз она увидит пролетающий мимо её окна мяч (считая, когда он пролетает и вниз и вверх)?

Три условия должны соблюдаться для чистоты эксперимента:
  • h должна быть больше 0
  • bounce больше 0, но меньше 1
  • уровень окна мамы меньше, чем h
Если все три условия соблюдаются, верните результат в виде целого числа, если нет — верните –1.

Например:

h = 3, bounce = 0.66, window = 1.5, результат = 3
h = 3, bounce = 1, window = 1.5, результат = -1
Решение на JavaScript 
bounce = (h, bounce, wndw) => {
    if (h > 0 && 1 > bounce > 0 && wndw < h) {
        res = 0;
        for (; h >= wndw; h *= bounce) {res += 1};
        return res*2 - 1;
    }
    return -1;
}

console.log(bounce(3, 0.66, 1.5));

28. Комплементарность ДНК

В дезоксирибонуклеиновой кислоте (ДНК) символы A и T, а так же C и G комплементарны друг другу. У вас есть функция с одной спиралью ДНК, вам нужно получить комплементарную сторону.

Например:

DNA_strand ("GTAT") = "CATA"
Решение на JavaScript 
DNA_strand = DNA => {
    res = '';
    for(elem in DNA) {
        switch (DNA[elem]){
            case 'A': res += 'T'; break;
            case 'C': res += 'G'; break;
            case 'G': res += 'C'; break;
            case 'T': res += 'A'; break;
            default: 0;
        }
    }
    return res;
}

console.log(DNA_strand("ATTGC"));

"indivisibility" == 1 // "i" встречается шесть раз

    obj = {};
    a.forEach((elem) => {
        obj[elem] == undefined ? obj[elem] = 1 : obj[elem] += 1;
    });
    ctr = 0;
    for (key in obj) {obj[key] > 1 ? ctr++ : 0}
    return ctr;
}

34. Выборочная сортировка

Например:

 Решение на JavaScript 
sort_array = a => {
    odd = a.filter(num => num % 2 != 0).sort();
    i = 0;
    for (elem in a) {
        if (a[elem] % 2 != 0) {a[elem] = odd[i]; i++};
    }
    return a;
}
console.log(sort_array([5, 3, 2, 8, 1, 4]));

40. Кошачий возраст? Собачий возраст!

У меня есть кот и собака. Я завёл их в одно и то же время. Это было humanYears (человеческих лет) назад.

Верните их текущий возраст как массив [humanYears, catYears, dogYears]. Если возраст кота и собаки рассчитывается по следующему правилу:

Кошачьи года:
  • +15 кошачьих лет за первый год
  • +9 кошачьих лет за второй год
  • +4 кошачьих года за каждый следующий год
Собачьи года:
  • +15 собачьих лет за первый год
  • +9 собачьих лет за второй год
  • +5 собачьих лет за каждый следующий год
Решение на JavaScript 
count_years = humanYearsAgo => {
    catYears = 0;
    dogYears = 0;
    switch (humanYearsAgo) {
        case 1:
            catYears = 15;
            dogYears = 15;
            break;
        case 2:
            catYears = 24;
            dogYears = 24;
            break;
        default:
            catYears = 24 + (humanYearsAgo-2) * 4;
            dogYears = 24 + (humanYearsAgo-2) * 5;
    };
    return [humanYearsAgo, catYears, dogYears];
};
console.log(count_years(3));

Возьмите число и просуммируйте его цифры, возведённые в степени и… Эврика!

Число 89 — первое двузначное число, для которого работает следующее правило:

Следующим таким числом является число 135:

Нужно написать функцию, которая будет искать числа, удовлетворяющие данному правилу в диапазоне от a до b (включительно).

Например:

sum_dig_pow(80, 135) == 89, 135
Решение на JavaScript 
sum_dig_pow = (a, b) => {
    for(; a <= b; a++) {
        num_of_digits = ("" + a).length;
        cur_a_arr = Array.from(a + '');
        res = 0;
        for(i = 1; i <= num_of_digits; i++) {
            res += Math.pow(+cur_a_arr[i - 1], i);
        };
        if (res == a) {console.log(res)};
    }
}
sum_dig_pow(80, 140);

reverse = a => {

    return(a);
};

sequenceSum(2,2,2) === 2

Решение на JavaScript 
sequenceSum = (a, b, c) => {
    for(res = 0; a <= b; a += c) {res = res + a};
    console.log(res);
}
sequenceSum(1,5,1);

45. Чарли и фотографии

Алиса и Боб были на выходных. Каждый из них сделал множество фотографий мест, в которых они были и теперь хотят показать Чарли всю коллекцию. Чарли не очень любит смотреть фотографии, тем более, что большинство фотографий похожи. Ему бы не хотелось смотреть на фотографию Эйфелевой башни 40 раз. Он сказал, что готов посмотреть фотографии только в том случае, если в коллекции Алисы и Боба одно и то же место на фотографиях не будет встречаться чаще, чем n раз. Можете ли вы им помочь удалить из коллекции те числа, которые встретились уже более n раз?

Например:

delete_nth([1,1,1,1],2) == [1,1]
delete_nth([20,37,20,21],1) == [20,37,21]
 Решение на JavaScript 
delete_nth = (a, n) => {
    obj = {};
    for(i = 0; i < a.length; i++) {
        !obj[a[i]] ? obj[a[i]] = 1 : obj[a[i]] += 1;
        obj[a[i]] > n ? a[i] = "R" : 0;
    };
    a = a.filter(x => x != "R" );
    return a;
};
console.log(delete_nth([1,1,1,1],2));
console.log(delete_nth([20,37,20,21],1));
